# MYSQL: BÁSICO AO INTERMEDIÁRIO

> Aulas do Curso em Vídeo com total foco em MySQL

---

## Etapas do Aprendizado

- **Mundo 1** Fundamentos: Melhorando a estrutura do banco de dados
- **Mundo 2** Inserindo Dados na Tabela: INSERT INTO
- **Mundo 3** Alterando a Estrutura de Dados: ALTER TABLE e DROP TABLE
- **Mundo 4** Manipulando Linhas: UPDATE, DELETE e TRUNCATE
- **Mundo 5** Gerenciando Cópias de Segurança MySQL (Backup de Banco de Dados)
- **Mundo 6** Operação Essencial: SELECT
- **Mundo 7** Modelo Relacional
- **Mundo 8** JOIN: Inner Join, Left Join, Right Join e Full Join
